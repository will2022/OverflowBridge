## Run Locally

Clone the project

```bash
  git clone https://github.com/will2022/overflowbridge
```

Go to the project directory

```bash
  cd overflowbridge
```

Install dependencies

```bash
  npm install
```

Make sure to fill all properties of the `config.ts` and then compile the TypeScript

```bash
  tsc
```

Start the bot

```bash
  node ./dist/src/
```
